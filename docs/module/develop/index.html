Tools for Developers

Develop modules support testing and development of Vistle and modules. A workflow for processing an actual data set should not benefit from any of these modules. This category also comprises modules that are still being worked on.

C

CreateCelltree create a cell tree

CreateVertexOwnerList create vertex owner list

D

Dropbear start Dropbear SSH server on compute nodes

F

FilterNode filter data objects

G

Gendat generate scalar and vector test data on structured and unstructured grids

GenIsoDat generate test data for IsoSurface module

GridElementStatistics show characteristics of selected unstructured grid element

M

ManipulateGhosts modify ghost cell information

MiniSim small simulation to compare performance with other in situ interfaces

MpiInfo print MPI, CPU binding and shared memory info

O

ObjectStatistics print object statistics

P

PrintAttributes print object attributes

R

Replicate deprecated

S

ShowCelltree show cell tree

ShowGridIndex show the index of each vertex in the grid

T

TestCellSearch test celltree

TestDynamicPorts test port removal

TestInterpolation test interpolation

ToPolyhedra transform unstructured cells into polyhedral cells

TransformGrid shuffle and mirror axes of rectilinear grids

V

Validate validate data object integrity
