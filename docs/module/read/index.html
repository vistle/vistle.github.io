Input Data from Files

Read modules read in data from files. They are the most common modules to start a pipeline strand in a workflow. Most often, they provide data that has not yet undergone the mapping stage. Together with the Simulation modules, these are the sources of the data flow network describing the workflow.

L

LoadCover instruct COVER to load a file

R

ReadCovise read COVISE data

ReadCoviseDirectory read COVISE data in a directory

ReadCsv read .CSV tables

ReadDuisburg Read Duisburg files

ReadDyna3D read LS-Dyna3D ptf files

ReadEnsight read EnSight Gold data

ReadFoam read OpenFOAM data

ReadIagNetcdf Read IAG data in NetCDF format

ReadIagTecplot read IAG Tecplot data (hexahedra only)

ReadIewMatlabCsvExports read a line or row from a csv file

ReadItlrFs3d read ITLR FS3D (Free Surface 3D) binary data

ReadModel read polygonal geometry models with Assimp (STL, OBJ, 3DS, Collada, DXF, PLY, X3D, ‚Ä¶)

ReadMpasPnetcdf Read MPAS files

ReadMpas Read MPAS files

ReadNek5000 read .nek5000 files

ReadSoundPlan Read SoundPLAN files.

ReadTsunami read ChEESE Tsunami files

ReadVtk read VTK data

ReadWrfChem read WRF-Chem (Weather Research and Forecasting model coupled to Chemistry) data files
