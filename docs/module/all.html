All Modules

A

AddAttribute (General) add attributes

ApplyTransform (Geometry) compute transformed coordinates

Assemble (General) assemble blocks of indexed geometry to larger blocks

AttachGrid (General) attach underlying grid to mappable data

AttachNormals (General) attach normals to grid

AttachShader (General) attach GLSL shader for use in COVER

B

BlenderRenderer (Render) Blender remote renderer

BoundingBox (Map) compute bounding boxes

C

COVER (Render) VR renderer for immersive environments

Cache (General) cache input objects

Calc (General) compute with coordinates and data

CellSelect (Filter) filter elements according to mapped data

CellToVert (Filter) convert cell mapped data to vertice mapped

CellToVertVtkm (Filter) convert cell mapped data to vertice mapped

Chainmail (Geometry) transform 2D data(triangle and quads) to rings

ClipVtkm (General) clip geometry at surface (CutGeometry)

Color (Map) map scalar and vector data to colors

ColorAttribute (General) specify solid object color as an attribute

ColorMetapostPart (Map) add color attribute from METApost description

ColorRandom (Map) map index data to random colors

CreateCelltree (Develop) create a cell tree

CreateCuboids (Geometry) creates cuboids out of a grid with (3D vector) mapped data where the grid’s coordinates are the cuboid centers and its mapped data the edge lengths in x, y, z

CreateVertexOwnerList (Develop) create vertex owner list

CutGeometry (Geometry) clip geometry at basic geometry like plane, cylinder or sphere

CuttingSurface (Map) cut through grids with basic geometry like plane, cylinder or sphere

CuttingSurfaceVtkm (Map) cut through grids with basic geometry like plane, cylinder or sphere

D

DelaunayTriangulator (Geometry) combine points to structured grid based on 3D Delaunay Triangulation

Displace (Filter) move vertices according to mapped data of first input

Distribute (General) broadcast input objects to all ranks

DomainSurface (Map) show surface of grid

Dropbear (Develop) start Dropbear SSH server on compute nodes

E

EnableTransparency (General) advise COVER to enable depth sorting

ExtractGrid (General) extract the grid a data object is mapped to

Extrema (Information) compute extremal values

F

FieldToLines (UniViz) field to lines

FilterCells (Filter) filter out some cells

FilterNode (Develop) filter data objects

FlattenTriangles (Geometry) flatten indexed triangle and quad geometry

FlowTopo (UniViz) flow topology

G

GenIsoDat (Develop) generate test data for IsoSurface module

Gendat (Develop) generate scalar and vector test data on structured and unstructured grids

GhostCellGenerator (General) Generate Ghost Cells for UnstructuredGrids

GridElementStatistics (Develop) show characteristics of selected unstructured grid element

I

InSituModule (Simulation) acquire data from SENSEI or Catalyst II instrumented simulations

IndexManifolds (Map) extract lower dimensional surfaces, lines or points from structured grids

IsoHeightSurface (Map) extract surface at constant height

IsoSurface (Map) extract surfaces with constant data level

IsoSurfaceVtkm (Map) extract surfaces with constant data level

L

LibSimController (Simulation) acquire data from LibSim instrumented simulations

LoadCover (Read) instruct COVER to load a file

M

ManipulateGhosts (Develop) modify ghost cell information

MapDrape (Filter) map coordinates and drape to height field

MetaData (General) transform meta data to mappable data

MiniSim (Develop) small simulation to compare performance with other in situ interfaces

MpiInfo (Develop) print MPI, CPU binding and shared memory info

O

ObjectStatistics (Develop) print object statistics

OsgRenderer (Render) OpenSceneGraph remote renderer

P

PrintAttributes (Develop) print object attributes

PrintMetaData (Information) print meta data

R

ReadCovise (Read) read COVISE data

ReadCoviseDirectory (Read) read COVISE data in a directory

ReadCsv (Read) read .CSV tables

ReadDuisburg (Read) Read Duisburg files

ReadDyna3D (Read) read LS-Dyna3D ptf files

ReadEnsight (Read) read EnSight Gold data

ReadFoam (Read) read OpenFOAM data

ReadIagNetcdf (Read) Read IAG data in NetCDF format

ReadIagTecplot (Read) read IAG Tecplot data (hexahedra only)

ReadIewMatlabCsvExports (Read) read a line or row from a csv file

ReadItlrFs3d (Read) read ITLR FS3D (Free Surface 3D) binary data

ReadModel (Read) read polygonal geometry models with Assimp (STL, OBJ, 3DS, Collada, DXF, PLY, X3D, …)

ReadMpas (Read) Read MPAS files

ReadMpasPnetcdf (Read) Read MPAS files

ReadNek5000 (Read) read .nek5000 files

ReadSoundPlan (Read) Read SoundPLAN files.

ReadTsunami (Read) read ChEESE Tsunami files

ReadVtk (Read) read VTK data

ReadWrfChem (Read) read WRF-Chem (Weather Research and Forecasting model coupled to Chemistry) data files

Replicate (Develop) deprecated

S

Sample (Filter) sample data on points, unstructured and uniform grids to a uniform grid

ScalarToVec (Filter) combine up to three scalar fields to a vector field

ShowCelltree (Develop) show cell tree

ShowGrid (Map) show outlines of grid cells

ShowGridIndex (Develop) show the index of each vertex in the grid

SortBlocks (General) sort data objects according to meta data

SpheresOverlap (Geometry) create lines between overlapping spheres

SplitDimensions (Filter) separate grid into 1D, 2D and 3D components

SplitPolyhedra (Geometry) convert polyhedral into simple cells

Statistics (UniViz) statistics

T

TestCellSearch (Develop) test celltree

TestDynamicPorts (Develop) test port removal

TestInterpolation (Develop) test interpolation

Thicken (Geometry) transform lines to tubes or points to spheres

Threshold (Filter) filter elements according to mapped data

ToPoints (Geometry) reduce geometry to underlying points

ToPolyhedra (Develop) transform unstructured cells into polyhedral cells

ToTriangles (Geometry) decompose polygons to triangles

ToUnstructured (Filter) transform structured grids to equivalent unstructured grid

Tracer (Map) compute particle traces and streamlines

Transform (General) apply transformation matrix to grid

TransformGrid (Develop) shuffle and mirror axes of rectilinear grids

V

Validate (Develop) validate data object integrity

Variant (General) add variant attribute

VecToScalar (Filter) transform vector fields to scalar fields

VectorField (Map) create lines from mapped vector data

VertToCellVtkm (Filter) convert vertex mapped data to cell mapped

VortexCores (UniViz) vortex corelines

VortexCriteria (UniViz) vortex criteria

W

WeldVertices (General) weld vertices and build indexed geometry
