Manipulate Data at Any Pipeline Stage

General modules are able to process data at any stage in the pipeline. Generally, they provide output for the same pipeline stage.

A

AddAttribute add attributes

Assemble assemble blocks of indexed geometry to larger blocks

AttachGrid attach underlying grid to mappable data

AttachNormals attach normals to grid

AttachShader attach GLSL shader for use in COVER

C

Cache cache input objects

Calc compute with coordinates and data

ClipVtkm clip geometry at surface (CutGeometry)

ColorAttribute specify solid object color as an attribute

D

Distribute broadcast input objects to all ranks

E

EnableTransparency advise COVER to enable depth sorting

ExtractGrid extract the grid a data object is mapped to

G

GhostCellGenerator Generate Ghost Cells for UnstructuredGrids

M

MetaData transform meta data to mappable data

S

SortBlocks sort data objects according to meta data

T

Transform apply transformation matrix to grid

V

Variant add variant attribute

W

WeldVertices weld vertices and build indexed geometry
