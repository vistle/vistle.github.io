Gendat

generate scalar and vector test data on structured and unstructured grids

Isosurface Example

This example uses the IsoSuface module to generate a surface with the data value 1. To set this up create the pipeline as described in the following picture and change the isovalue parameter of the Surface module to 1.

Alternatively load vistle/examples/gendat-isosurface.vsl.
